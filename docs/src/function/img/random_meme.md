# 随机表情包

## 概述

本功能用于随机各种表情包

## 指令

### /随机表情包

- 指令： `/随机表情包 [页数]`

- 参数：

  - `页数`：表情包页数，可选，默认为 1

- 详情：

  获取随机表情包菜单
