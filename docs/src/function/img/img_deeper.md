# 图片智慧功能

:::tip
由于腾讯的要求，本功能在 `QQ官方` 平台上暂不可用
:::

## 概述

本功能提供了图片智慧相关的功能

## 指令

:::tip
图片参数可以跟在指令后，也可以分开发送——先发送指令后发送图片
:::

### tag识别

- 指令： `tag识别`
- 别名： `/tag识别`

- 参数：

  - `图片`： 需要识别的图片

- 详情：

  识别图片内元素（一般用于二次元图片），提供了识别图片内标签的功能，可用于找到你所需的 tag，并用于写作、AI 绘画等功能

### 去背景

- 指令： `/去背景`
- 别名： `/removebg`

- 参数：

  - `图片`： 需要去背景的图片

- 详情：

  去除图片内背景，只保留人像（一般用于二次元图片）
