# HTTP猫猫/fumo

## 概述

每一张猫猫/fumo都代表着一个 [HTTP状态码](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Reference/Status)

## 指令

### /httpcat

- 指令： `/httpcat [状态码]`

- 参数：

  - `状态码`： HTTP状态码

- 详情：

  找一只HTTP猫猫

### /随机httpcat

- 指令： `/随机httpcat`

- 详情：

  随机一只HTTP猫猫

### /httpfumo

- 指令： `/httpfumo [状态码]`

- 参数：

  - `状态码`： HTTP状态码

- 详情：

  来一只HTTP Fumo

### /随机httpfumo

- 指令： `/随机httpfumo`

- 详情：

  随机一只HTTP Fumo
