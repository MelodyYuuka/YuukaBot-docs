# 塔罗牌相关

## 指令

### 抽塔罗牌

- 指令： `/抽塔罗牌`

- 别名： `抽塔罗牌`, `/抽东方塔罗牌`, `抽东方塔罗牌`

- 详情：

  抽取一张大阿卡纳牌（东方Project风格）

### 抽ba塔罗牌

- 指令： `/抽ba塔罗牌`

- 别名： `抽ba塔罗牌`

- 详情：

  抽取一张大阿卡纳牌（蔚蓝档案风格）

### 抽塔罗牌阵

- 指令： `/抽塔罗牌阵 [牌阵名称]`

- 别名： `抽塔罗牌阵`, `抽牌阵`, `/抽牌阵`

- 参数：

  - `牌阵名称`：可选，指定要抽取的牌阵名称，不填则随机选择一个牌阵，支持的牌阵可以通过 [/塔罗牌阵列表](#塔罗牌阵列表) 查看

- 详情：

  抽取一个指定牌阵的塔罗牌组进行占卜

### 塔罗牌阵列表

- 指令： `/塔罗牌阵列表`

- 别名： `塔罗牌阵列表`, `塔罗阵型列表`, `/塔罗阵型列表`

- 详情：

  查看支持的塔罗牌阵列表
