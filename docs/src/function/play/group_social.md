# 群发言排行

:::tip
该功能已注册 **推送服务**，详见 [服务管理](../admin/task_manager.md)
:::

## 概述

本功能用于统计群内成员的发言数量，查看今日或昨日的发言排行榜，看看谁才是唠嗑大王~

## 指令

### 排行菜单

- 指令： `/本群排行菜单`

- 别名： `/本群排名菜单`、`/群排行菜单`、`/群排名菜单`

- 详情：

  查看本群排行菜单，提供今日和昨日发言排行的快捷入口

### 今日发言排行

- 指令： `/本群今日发言排行`

- 别名： `/本群发言排行`

- 参数：

  - `页码`： 可选参数，指定查看的页码，默认为第1页，每页显示10条

- 详情：

  查看本群今日发言排行

### 昨日发言排行

- 指令： `/本群昨日发言排行`

- 参数：

  - `页码`： 可选参数，指定查看的页码，默认为第1页，每页显示10条

- 详情：

  查看本群昨日发言排行

## 定时推送

本功能支持定时推送服务，每天 **9:00** 自动向已开启推送的群发送 **昨日群发言排行**。

## 支持平台

| 平台 | 支持情况 |
| ---- | -------- |
| QQ官方Bot | ✅ 支持 |
| QQ频道Bot | ✅ 支持 |
| Kook | ✅ 支持 |

## 注意事项

- 该功能仅在群聊中可用，私聊使用会提示前往群内查看
- QQ官方Bot 需开启全量消息才能完整统计发言数量
