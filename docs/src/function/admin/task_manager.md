# 任务/主动功能管理

:::info
本功能在 `QQBot` 与 `KOOK` 平台上尚在测试，有问题可以及时报告
:::

## 概述

YuukaBot-v6 对于自动推送信息与高频主动功能额外提供了 **任务/主动功能管理** 来便捷各群个性化使用。

## 指令

### 开启任务 <Badge type="tip" text="群管或群主" vertical="top" />

- 指令： `开启任务[任务/主动功能名]`

- 参数：
  - `任务/主动功能名`： 需要开关的任务名

- 详情：

  开启对应任务

### 关闭任务 <Badge type="tip" text="群管或群主" vertical="top" />

- 指令： `关闭任务[任务/主动功能名]`

- 参数：
  - `任务/主动功能名`： 需要开关的任务名

- 详情：

  关闭对应任务

### 任务列表

- 指令： `任务列表`

- 详情：

  查看任务列表及本群开关状态

### 查看任务

- 指令： `查看任务[任务/主动功能名]`

- 参数：

  - `任务/主动功能名`： 需要的任务名

- 详情：

  查看任务详情
