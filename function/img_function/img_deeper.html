# 图片标签识别

# 概述

本功能提供了识别图片内标签的功能，可用于找到你所需的 tag，并用于写作、AI 绘画等功能

# 指令

  • 指令： tag识别

  • 参数：

    • 图片： 需要识别的图片

  • 详情：

    识别图片内元素（一般用于二次元图片）

提示

图片参数可以跟在指令后，也可以分开发送——先发送指令后发送图片
